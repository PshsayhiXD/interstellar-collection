# Pshsayhi's Interstellar Collection

Welcome to the **Interstellar Collection** by PshsayhiXD! This repository serves as the definitive home for a curated collection of premium gameplay and visual mods designed for the Interstellar web game environment. 

At the core of this collection sits the flagship **Pshsayhi Mod Loader**, a custom, dynamic GUI overlay that seamlessly injects into the game client. It empowers players to elegantly load, configure, and manage all their installed Interstellar mods and cosmetic themes on the fly. Accompanying the Mod Loader is a suite of bundled example mods (ranging from subtle visual adjustments to dynamic mini-pets) showcasing the power of the loader's framework.

---

## Building & Packaging

The repository includes a custom, zero-friction packaging script (`build.js`) powered by the Archiver library. It quickly compresses any mod directory into a deployment-ready `.zip` file for the game engine.

### Requirements
You'll need `Node.js` installed to build the mods locally. First, initialize the dependencies:
```bash
npm install
```

### Usage
To easily package an existing mod or the Mod Loader, use the npm run scripts. 

For example, to package the Mod Loader itself:
```bash
npm run zip -- pshsayhi-mod-loader
```
The script will instantly output the deployment-ready `.zip` into the root directory of your workspace.

---

## Installation

1. Grab the latest `.zip` generated by the build script (or from the repository's Releases page).
2. Load the `.zip` into your Interstellar Mod Manager interface.
